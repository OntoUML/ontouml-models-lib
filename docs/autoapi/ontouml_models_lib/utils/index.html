ontouml_models_lib.utils

The utils module provides utility functions and helpers for the OntoUML/UFO catalog package.

This module contains auxiliary functions that support the core functionality of the package, such as file handling, data processing, and other common tasks that are used across different components of the library. The utilities here are intended to simplify and streamline operations within the package.

Note:

This module is typically not accessed directly by users. Its functions and classes are used internally by other components of the package.
